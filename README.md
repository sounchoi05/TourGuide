# TourGuide Android APP
여행 가이드 (Android APP)

https://spain.changki.com
이 사이트의 결과와 최대한 유사하게 만든 Android APP

이 앱은 사용자가 직접 여행 정보를 구성하고 현지에서 유용하게 활용할 수 있도록 제작된 맞춤형 종합 여행 가이드 앱입니다. 
주요 특징은 다음과 같습니다.

1. 완벽한 사전 계획: 관리자 모드를 통해 세부 일정, 숙소, 이동 경로, 주차장 정보를 등록하고 구글맵과 연동하여 길 찾기를 돕습니다.
2. 스마트한 입력: 웹에서 복사한 HTML 텍스트만으로 맛집과 스팟 리스트가 자동 등록되며, 구글 API 기반의 회화표현 자동 번역을 지원합니다.
3. 현지 밀착형 가이드: 다국어 TTS를 지원하는 오디오 가이드와 상황별 회화 탭이 탑재되어 언어 장벽을 낮춰줍니다.
4. 편리한 유틸리티: 실시간 날씨와 환율을 한눈에 확인하고, 다중 화폐가 지원되는 가계부로 꼼꼼한 지출 관리가 가능합니다.
5. 데이터 안전성: 오프라인 환경에서도 작동하도록 설계되었으며, 모든 여행 데이터와 사진은 하나의 ZIP 파일로 백업 및 복원이 가능합니다.
